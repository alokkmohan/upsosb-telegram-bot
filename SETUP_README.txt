====================================================
  UPSOSB TELEGRAM BOT - SETUP GUIDE
====================================================

ZAROORAT:
  - Node.js v18 ya usse upar (https://nodejs.org)

----------------------------------------------------
WINDOWS SERVER PE CHALANE KA TARIKA:
----------------------------------------------------
1. Pendrive se folder ko server pe copy karo
2. "start.bat" file par double-click karo
3. Bot shuru ho jayega

----------------------------------------------------
LINUX SERVER PE CHALANE KA TARIKA:
----------------------------------------------------
1. Pendrive se folder ko server pe copy karo
   (ya terminal mein yeh command chalao):

   cp -r /media/pendrive/upsosb-telegram-bot /home/

2. Terminal mein folder ke andar jao:
   cd /home/upsosb-telegram-bot

3. Script ko permission do (sirf pehli baar):
   chmod +x start.sh

4. Bot chalao:
   ./start.sh

----------------------------------------------------
BOT BAND KARNE KA TARIKA:
----------------------------------------------------
  Ctrl + C dabao

----------------------------------------------------
PROBLEM AAYE TO:
----------------------------------------------------
  Node.js install karo:

  Windows: https://nodejs.org se download karo

  Linux:
    sudo apt update
    sudo apt install -y nodejs npm
