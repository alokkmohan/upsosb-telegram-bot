# Mathematics Study Material

## File Structure

Place your PDF files in this folder with the following names:

### Complete Book
- `complete-book.pdf` - Full mathematics textbook

### Chapter-wise PDFs
1. `chapter-1.pdf` - अध्याय 1 - वास्तविक संख्याएँ (Real Numbers)
2. `chapter-2.pdf` - अध्याय 2 - बहुपद (Polynomials)
3. `chapter-3.pdf` - अध्याय 3 - दो चरों वाले रैखिक समीकरण (Linear Equations)
4. `chapter-4.pdf` - अध्याय 4 - द्विघात समीकरण (Quadratic Equations)
5. `chapter-5.pdf` - अध्याय 5 - समांतर श्रेणी (Arithmetic Progressions)
6. `chapter-6.pdf` - अध्याय 6 - त्रिभुज (Triangles)
7. `chapter-7.pdf` - अध्याय 7 - निर्देशांक ज्यामिति (Coordinate Geometry)
8. `chapter-8.pdf` - अध्याय 8 - त्रिकोणमिति (Trigonometry)
9. `chapter-9.pdf` - अध्याय 9 - त्रिकोणमिति के अनुप्रयोग (Applications of Trigonometry)
10. `chapter-10.pdf` - अध्याय 10 - वृत्त (Circles)
11. `chapter-11.pdf` - अध्याय 11 - रचनाएँ (Constructions)
12. `chapter-12.pdf` - अध्याय 12 - क्षेत्रमिति (Areas Related to Circles)
13. `chapter-13.pdf` - अध्याय 13 - पृष्ठीय क्षेत्रफल और आयतन (Surface Areas and Volumes)
14. `chapter-14.pdf` - अध्याय 14 - सांख्यिकी (Statistics)
15. `chapter-15.pdf` - अध्याय 15 - प्रायिकता (Probability)

## How Bot Works

When user clicks on a chapter button:
- Bot checks if PDF file exists
- If exists: Sends the PDF to user
- If not exists: Shows "Coming soon" message

## Adding PDFs

Simply copy your PDF files to this folder with the correct names mentioned above.
